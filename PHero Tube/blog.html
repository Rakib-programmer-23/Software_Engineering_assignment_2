Var :

Variables declared with `var` are function-scoped, meaning they are only accessible within the function where they are defined.

let :

Let is blocked scope that means it can't be access from outer scope but you can still re-assign the value.

const :

const variables are only accessible within their declared block. Once you assign a value to a const variable, you cannot change it later.

Null :

Null represents the intentional absence of a value and Null is used as a placeholder value for missing data.

Undefined :

Undefined is the default value for uninitialized variables and Undefined is returned when trying to access a non-existent property of an object.

REST API

=>REST API stands for Representational State Transfer API. It is a software architectural style for designing web services.
=>APIs are often used to develop the backend functionality of websites and mobile apps, providing access to data and resources.
